# Tether

An ADHD-friendly life management app built around crash states, households, and real-world constraints.

## What Tether Does

- Tracks crash vs survival vs functional days
- Generates small, time-boxed missions instead of giant to-do lists
- Connects cleaning, food, sleep, and work into one system
- Designed for households, not just individuals

## Privacy First

- Anonymous by default
- No ads, no data selling, ever
- One-tap “delete my data” inside the app
